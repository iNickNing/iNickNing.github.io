Hexo文章发布备忘

平时习惯在自己的笔记本上面记录,较少发布文章到外面的环境

Hexo 的使用也相对次数变少了

下面记录一下一些常用的命令

当然,直接访问官方文档也可以: https://hexo.io/zh-cn/docs/commands

常用命令

创建新的文章: new

新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

生成页面静态文件: generate

将我们新建的文章编译一下,变成html文件,用于发布

本地启动服务: server

用于查看文章的情况,主要是发布之前的预览

文章发布: publish

没感受到它可以做什么

文章部署: deploy

部署站点,目前配置了 GitHub

Linux(CentOS7)下载yum下的rpm包
